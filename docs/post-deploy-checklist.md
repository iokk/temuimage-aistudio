# 上线后验证清单

这份清单用于每次 Zeabur / 服务器重新部署后，快速验证系统是否处于可用状态。

## A. 基础可达性

1. 打开首页，确认页面能正常加载，没有白屏或无限转圈
2. 检查健康接口：

```bash
curl http://<your-domain-or-host>/_stcore/health
```

期望返回：

```text
ok
```

3. 确认顶部显示最近更新日期和 4 个核心能力摘要
4. 确认左侧导航可点击：
   - `工作台`
   - `批量出图`
   - `快速出图`
   - `标题优化`
   - `图片翻译`

## B. 管理员工具模式（无 DATABASE_URL 时）

1. 登录页应显示：`当前为管理员模式`
2. 不应再暴露团队用户注册、钱包、项目、兑换码等入口
3. 使用管理员密码登录后，应能直接进入后台
4. 后台里至少能看到：
   - `基础设置`
   - `密码配额`
   - `API Key管理`
   - `合规设置`
   - `提示词`
   - `标题模板`
   - `组图模板`

## C. 系统配置检查

进入后台 `API Key管理` 后确认：

1. 系统 Gemini Key 列表可见、可启用/禁用/删除
2. 系统中转站配置可见并可保存：
   - `API 地址`
   - `默认图片模型`
   - `分析/标题模型`
   - `API Key`
3. 保存后刷新页面，配置仍然存在

推荐系统中转站配置：

```text
API 地址: https://newapi.aisonnet.org/v1
默认图片模型: seedream-5.0 或 gemini-3.1-flash-image-preview
分析/标题模型: gemini-3.1-flash-image-preview
```

## D. 我的凭据检查

登录页 `我的凭据` 中确认：

1. 可以填自己的 Gemini Key
2. 可以填自己的中转站：
   - URL
   - Key
   - 模型
3. 点“使用我的 Gemini Key”或“使用我的中转站”后，不报错且能进入系统

## E. 功能可用性检查

### 1 批量出图

1. 上传商品图后，“AI分析商品”按钮应可点击
2. 若按钮不可点击，页面应直接显示原因提示
3. 选择类型后，“AI生成图需文案”应可点击
4. 完成图需后，“确认开始生成”应可点击
5. 生成后应看到：
   - 结果摘要卡
   - 成功/失败数
   - 标题结果（如果开启）

### 2 快速出图

1. 上传图片、填写商品名、选择类型后，“开始生成”应可点击
2. 若不可点击，页面应直接显示原因提示
3. 生成后应看到结果摘要卡和图片结果

### 3 标题优化

1. 文字模式：输入足够商品信息后，生成按钮应可点击
2. 图片模式：上传图片后，生成按钮应可点击
3. 若不可点击，页面应显示具体原因
4. 结果区应出现：
   - 标题结果摘要卡
   - 标题内容
   - 警告/错误说明（如有）

### 4 图片翻译

1. 上传图片后，“开始处理”或“提交后台任务”应可点击
2. relay 模式下：
   - 如果当前模型支持翻译出图，应可正常执行
   - 如果不支持，应明确提示模型能力不足
3. 推荐 relay 图片翻译出图模型：`seedream-5.0`
4. 推荐 relay 文本分析模型：`gemini-3.1-flash-image-preview`
5. 结果区应出现：
   - 结果摘要卡
   - 译后图 / 文本结果
   - 错误/警告说明

## F. 中转站能力验证

建议至少验证以下模型：

```text
gemini-3.1-flash-image-preview
seedream-5.0
seedream-4.6
z-image-turbo
```

推荐当前使用策略：

- 图片生成主力：`seedream-5.0` 或 `gemini-3.1-flash-image-preview`
- 分析 / 标题：`gemini-3.1-flash-image-preview`
- 图片翻译出图：优先 `seedream-5.0`

## G. 问题排查顺序

如果上线后“不能用”，按这个顺序看：

1. `/_stcore/health` 是否返回 `ok`
2. 管理员是否能登录后台
3. 系统中转站 Key / URL 是否已保存
4. 当前模型是否支持当前任务
5. 页面是否显示了“为什么不能点”的原因提示
6. 浏览器是否需要强刷缓存

## H. 每次上线后的最小确认

如果你时间很紧，至少做这 8 项：

1. 健康检查 `ok`
2. 管理员能登录
3. 后台能打开
4. 系统中转站配置仍在
5. `1 批量出图` 能进并可点击分析
6. `2 快速出图` 能生成
7. `3 标题优化` 能出标题
8. `4 图片翻译` 能处理，或明确提示模型不支持
