# UI Refresh Design

## Goal

Simplify the current Streamlit interface into a clearer professional operations dashboard with a unified workbench, calmer admin-mode messaging, and more consistent module structure.

## Scope

- Reduce the visual weight of database warnings when administrator mode is already usable.
- Introduce a `工作台` landing page.
- Normalize navigation for the four core tools.
- Improve top-level hierarchy, cards, spacing, and sidebar grouping.

## Direction

Use a professional back-office visual language: blue-green primary palette, structured status cards, compact but legible spacing, and consistent iconography. Keep the current product as a Streamlit tool rather than forcing a fake website shell.

## Constraints

- No new product scope.
- No dependency on team database for basic admin-mode usability.
- Preserve all existing page functions while improving framing and information architecture.
